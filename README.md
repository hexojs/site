The website for Hexo.

## Getting started

Install dependencies:

``` bash
$ git clone https://github.com/hexojs/site.git
$ cd site
$ npm install
```

Generate:

``` bash
$ hexo generate
```

Run server:

``` bash
$ hexo server
```
